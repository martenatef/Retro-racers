# Retro Racers (HTML5 Local Multiplayer)

Top‑down 2‑player racing game built with pure HTML5 canvas and JavaScript.  
Features custom car physics (drift, off‑road, collisions), dynamic camera, nitro system, lap/checkpoint logic starting from 0, replay, particles, and basic sound effects.

▶ **Play in browser:** https://martenatef.github.io/Retro-racers/

---

## Features

- Local 2‑player racing on the same keyboard  
- Custom car physics: acceleration, friction, drift, off‑road slowdown  
- Dynamic camera that follows both players and zooms in/out  
- Lap & checkpoint system (laps start from 0 and only increment after a full valid lap)  
- Nitro system with HUD bars for both players  
- Basic SFX (engine loop, start, crash, nitro)  
- Multiple hand‑crafted tracks with different layouts  
- Simple replay of the last race

---

## Controls

**Player 1**

- Move: **W / A / S / D**  
- Drift: **Space**  
- Nitro: **Left Shift**

**Player 2**

- Move: **Arrow keys**  
- Drift / Nitro: **Right Shift**  
- Pause: **P**

---

## Tech Stack

- HTML5 Canvas  
- Vanilla JavaScript  
- Simple custom physics & camera system  
- No external game engine
